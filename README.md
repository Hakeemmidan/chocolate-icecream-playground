# _
Just a demo app for testing some things

## Setup
1. `bundle install`
2. `rails db:create`
3. `rails db:migrate`
4. `rails db:seed`

## Start
1. `rails s`
2. visit http://localhost:3000

## Associations
- A product has many reviews
- A review belongs to a product
